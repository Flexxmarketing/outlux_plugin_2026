# Flexxmarketing Client Plugin

A custom WordPress plugin developed by [Flexxmarketing](https://flexxmarketing.nl/) for use in client projects. This plugin extends WordPress functionality and integrates custom-built features tailored for Flexxmarketing's websites.

## Plugin Info

- **Name**: Flexxmarketing Client Plugin
- **Version**: 3.0.0
- **Author**: Duncan Heffron | [Flexxmarketing.nl](https://flexxmarketing.nl/) 
- **Requires at least**: WordPress 6.0.0
- **Tested up to**: WordPress 6.7.1
- **Requires PHP**: 8.1
- **Text Domain**: `flexx-client-plugin`

## Features

- Custom WPBakery Page Builder elements
- Reusable settings module for theme integration
- Modular class-based architecture
- Lightweight and optimized for Flexxmarketing themes

## Installation

1. Upload the plugin to your WordPress installation:
    - Via admin: go to **Plugins → Add New → Upload Plugin**
    - Or manually upload to `/wp-content/plugins/`
2. Activate the plugin via the Plugins menu in WordPress.

## Development

- Built to integrate with Flexxmarketing themes (e.g., Flexx Base Theme)
- Uses modern PHP structure under `classes/` folder
- Extendable and client-ready
